# Инструкцию по разворачиванию/установки на ОС.

- Создать ВО, и активировать его
- Загрузить `poetry`
    ```bash
    pip install poetry
    ```
-  Установить зависимости
    ```bash
    poetry install
    ```
# Описание параметров утилит с примерами использования.

Запустить сервер

```bash
python ./server.py
```

Запустить клиента, отправить сообщения на сервер

```bash
python ./client.py
```
